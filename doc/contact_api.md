### 关注用户
#### 请求
> POST /api/v1/contact/{user_id}/follow

#### 请求参数说明
| 参数 | 必要 | 说明 |
| ------------- | ------------- | ----- |
| user_id | 是 | 联系人 user_id |
----

### 取消关注
#### 请求

| 参数 | 必要 | 说明 |
| ------------- | ------------- | ----- |
| user_id | 是 | 联系人 user_id |
